3.17. LRB - Place of Gesture Articulation¶

3.17.1. Description¶

Specifies the appendage(s) used to perform the gesture

3.17.2. Format¶

entry ::=  (null | code) [op (null | code)]*
null  ::=  ""
code  ::=  "L" | "R" | "B" | "H" | "LF" | "RF" | "W"
op    ::=  "+" | "-" | "/" | "."

3.17.3. Values¶

Codes have the following interpretations:

Code Meaning
L Left hand
R Right hand
B Both hands
H Head
LF Left foot
RF Right foot
W Whole body

Ops have the following interpretations:

Op Meaning
+ Simultaneous gestures
- Sequential gestures without return to neutral position
/ Gestures morph into one another
. Sequential gestures with return to neutral position - DEPRECATED

3.17.4. Vertical Dependencies¶

See note on continued gestures below

3.17.5. Horizontal Dependencies¶

LRB is dependent on the Form Column.

LRB must have the same number of codes and the same number and type of ops as the Form entry on which it depends.

Note

If the code in Form is continued (ending with ~), the corresponding code in LRB must be the same as the LRB code associated with the Form code being continued on the immediately preceding line with a communicative act on the part of the speaker.

Example:

Line 1:
  Form: point + hold
  LRB: R + L
Line 2:
  Form: point~
  LRB: R
